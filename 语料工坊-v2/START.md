# 一键启动说明

## 推荐方式

双击项目根目录中的：

`启动语料工坊.cmd`

脚本会自动：

- 检查后端虚拟环境 `backend\.venv`
- 如果前端缺少 `node_modules`，自动执行 `npm install`
- 启动后端服务 `http://127.0.0.1:8765`
- 启动前端服务 `http://127.0.0.1:5173`
- 自动打开浏览器

## PowerShell 方式

在项目根目录运行：

```powershell
.\start.ps1
```

## 注意

启动后会出现两个命令行窗口，分别是后端和前端。使用软件期间不要关闭这两个窗口。
